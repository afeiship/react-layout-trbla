## usage
1. import css
  ```scss
  // or use sass
  @import '~@jswork/wsui-layout-trbla/dist/index.scss';
  @import "~@jswork/react-layout-trbla/dist/style.scss";

  // customize your styles:
  $react-layout-trbla-options: ()
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
